# OSINT Cyber Tool

Мощный инструмент для проведения автоматизированных OSINT-расследований с использованием рекурсивного движка корреляции данных. Построен на базе **FastAPI**, **WebSockets** и **SQLAlchemy**.

## 🚀 Основные возможности

* **Автоопределение типа входных данных:** система автоматически классифицирует запросы (IP, Телефон, Email, MAC-адрес, BTC/ETH кошелек, VIN автомобиля, Домен, Никнейм, CVE, Хэш, Координаты, Банковская карта, ИНН, ФИО и др.).
* **Рекурсивный движок корреляции:** автоматически извлекает новые зацепки (leads) из результатов работы модулей и запускает по ним новые проверки в рамках одной сессии (глубина рекурсии до 5 уровней).
* **Интеграция с популярными OSINT API и утилитами:**
  * Поиск по никнейму (`maigret`, `steam`, `github`, `reddit`, `twitch`, `pastebin` и др.)
  * Проверка почты на утечки (`holehe`, `gravatar`)
  * Сканирование IP и доменов (`shodan`, `censys`, `greynoise`, `abuseipdb`, `dns`, `ssl`, `wayback`)
  * Проверка телефонов, ИНН и судебных дел (модули для РФ и СНГ)
* **Интерактивный UI:** веб-интерфейс реального времени на WebSockets с логированием каждого шага.
* **База данных:** сохранение результатов сканирования в SQLite для последующего анализа и генерации досье.

## 🛠 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/GuinnesCoder/projectosint.git
cd projectosint
```

### 2. Настройка виртуального окружения
```bash
# Создание виртуального окружения
python -m venv .venv

# Активация окружения (Windows)
.venv\Scripts\activate

# Активация окружения (Linux/macOS)
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

### 3. Настройка окружения (`.env`)
Создайте файл `.env` в корневой папке проекта на основе ваших ключей доступа к API:
```env
# Пример настроек API (заполните нужные ключи)
SHODAN_API_KEY=your_key_here
CENSYS_API_ID=your_id_here
CENSYS_API_SECRET=your_secret_here
INTELX_API_KEY=your_key_here
INTELX_HOST=intelx.io
# И другие переменные окружения...
```

### 4. Запуск приложения
Запустите сервер разработки:
```bash
uvicorn main:app --reload
```
После запуска веб-интерфейс будет доступен по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000).

## 📂 Структура проекта

* `main.py` — основной файл приложения FastAPI, маршрутизация WebSockets и логика рекурсивного поиска.
* `database.py` и `models.py` — конфигурация базы данных SQLAlchemy и схема хранения Dossier.
* `modules/` — подключаемые OSINT-модули и обертки для внешних утилит.
* `templates/` и `static/` — файлы фронтенда (HTML/CSS/JS).
* `requirements.txt` — зависимости проекта.
